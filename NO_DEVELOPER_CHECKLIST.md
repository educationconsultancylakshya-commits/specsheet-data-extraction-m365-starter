# No-Developer Build Checklist

Use this checklist to build the solution without writing code.

## SharePoint

- [ ] Create SharePoint site
- [ ] Create document library: Spec Sheet Files
- [ ] Create list: Attribute Library
- [ ] Create list: Attribute Value Dictionary
- [ ] Create list: Extraction Jobs
- [ ] Create list: Extracted Values
- [ ] Create list: Final Results
- [ ] Create list: Audit Log
- [ ] Add indexed columns listed in docs

## Power Apps

- [ ] Create tablet canvas app
- [ ] Connect SharePoint lists
- [ ] Build Home screen
- [ ] Build Attribute Management screen
- [ ] Build Bulk Upload screen
- [ ] Build Spec Sheet Upload screen
- [ ] Build Validation Desk screen
- [ ] Build Results Dashboard screen
- [ ] Add formulas from power-apps/powerfx-snippets.md

## Power Automate

- [ ] Create job flow
- [ ] Create attribute import flow
- [ ] Create AI extraction flow
- [ ] Create Excel export flow
- [ ] Add error handling to each flow
- [ ] Test each flow manually

## AI Builder

- [ ] Configure OCR / text recognition
- [ ] Configure document extraction or prompt
- [ ] Test with 3 documents
- [ ] Check JSON output
- [ ] Confirm confidence scoring

## UAT

- [ ] Upload 3 spec sheets
- [ ] Review extracted values
- [ ] Approve/edit/reject values
- [ ] Export cleaned data
- [ ] Confirm missing values are visible
